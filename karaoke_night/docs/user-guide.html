Getting Started

Karaoke Night is a personal song finder. You load a session file containing your singers, categories, and songs, then use the app to find and play them on YouTube.

1
Open the app
Tap Let's Sing! on the landing page to open the song finder.
2
Load a session file
Go to Session → Open and choose your .csv session file. Or use Session → Open Sample to load the built-in example. To get a template to edit, use Session → Download → Blank Session.
3
Browse and sing
Expand a Singer, then a Category to see songs. Tap any song card to search YouTube.
Session Menu

The Session menu controls loading and managing your session. Some items are disabled until a session is loaded.

Session

NewBuild a session from scratch using the inline entry panel. Disabled once a session is loaded — use Add to append singers.
OpenLoad one or more .csv session files. Warns if a session is already loaded.
AddLoad one or more additional .csv files and append their singers to the current session.
ReopenOptionally drop singers, then opens the Add file picker. Useful mid-party.
Save As...Downloads the current session as a .csv — includes all edits, excludes dropped/deleted items.
CloseClears the session and returns to the welcome screen.
Open SampleLoads the built-in sample session from the server.
Download →Download Sample Session or Blank Session template to your device.
ExitReturns to the landing page.

Singer

Drop Singer(s)Removes selected singers from the session. Saved for Undrop.
Undrop Singer(s)Restores dropped singers. Disabled when nothing has been dropped.

Editing Your Session

Every singer header, category header, and song card shows a small ✏️ pencil icon. Tap it to open the edit menu for that item.

✏️ Edit Menu

RenameOpens an inline text field. Enter commits, Escape cancels.
DeleteRemoves the singer, category, or song from the session immediately.
Add Above / BelowSingers and categories — inserts a new item above or below. Enter adds, blank Enter exits.
Add Before / AfterSongs — inserts a new song before or after in the card grid. Same Enter flow.
💡 Tip: Use Save As... after editing to save your updated session as a .csv file.
Browsing Songs

Songs are organized in a Singer → Category → Song hierarchy. Tap any header to expand or collapse it.

1
Tap a Singer header to expand
Shows all categories for that singer. Tap again to collapse.
2
Tap a Category header to expand
Shows the song cards in that category.
3
Tap a song card
Opens a YouTube search for that song in the current search mode (Karaoke or Music Video).
4
Tap Back to return
Your scroll position, open sections, and search state are all restored automatically.
Searching

Two search tools are available in the top controls bar.

🔍
Filter
Searches your loaded session. Narrows singers and categories to matching songs only. Tap ✕ to clear and restore full view. Disabled until a session is loaded.
▶
Find on YouTube
Search YouTube directly for any song — no session needed. Type a song name and press Enter. Tap ✕ to clear.
💡 Tip: Use Filter to quickly find a song in your session list. Use Find on YouTube for ad hoc searches for songs not in your session.
YouTube Search Modes

The two mode buttons next to the Find on YouTube box control how songs are searched on YouTube.

🎤
Karaoke Mode (default)
Searches YouTube for karaoke [song title] — finds backing tracks with lyrics.
🎵
Music Video Mode
Searches YouTube for [song title] music video — finds the original artist recording.
💡 Tip: The active mode button appears lit/bright. Your mode selection is remembered if you navigate back from YouTube.
Session Files

A session file is a .csv file that defines your singers, categories, and songs. It can be created and edited in any spreadsheet app (Google Sheets, Excel, LibreOffice Calc) or a plain text editor.

File Format

Column 1# marks a comment row — entire row is ignored by the app
Column 2Singer name
Column 3Category name
Column 4Song title
Columns 5–6Artist and Full URL — reserved for future use
💡 One row can do multiple things: A single row can define a Singer, Category, and first Song all at once — just fill in columns 2, 3, and 4 together.
💡 File naming: Name your session files starting with session_ (e.g. session_friday.csv) to keep them organized. Use Session → Download → Blank Session to get a template to start from.
Troubleshooting
ProblemFix
Session won't loadMake sure the file is a .csv file. Check that it follows the column format — Singer in column 2, Category in column 3, Song in column 4.
No singers showingThe file may have no data rows. Open Session → Open Sample to verify the app is working, then check your file.
Filter not workingA session must be loaded first — the Filter is disabled until a session is open.
Scroll position lostMake sure you tap Back — don't close the YouTube tab. The app restores scroll position on Back navigation.
YouTube opens wrong searchCheck which mode is active — 🎤 Karaoke Mode or 🎵 Music Video Mode. The lit button is the active one.
Open Sample failsThis feature requires an internet connection — it fetches the sample file from the app server.
App looks wrong after updateHard refresh the page: Ctrl+Shift+R on desktop, or clear browser cache on mobile.
